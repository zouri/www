题目备注

1.关于防火墙配置

  1. 一定要在 permanent 状态下配置
  2. 如题目中未明确指出使用 drop 一律使用 reject
  3. 配置端口转发时如果没有明确说明 tcp 还是 udp 那么两个都配置上

2.关于 samba 共享文件夹

安全上下文 samba_share_t
挂载查看帮助使用命令: man mount.cifs

fstab 文件例子

//server/common /mnt/data ext4 defaults,username=rob,password=redhat,sec=ntlmssp,multiuser 0 0

3.关于 nfs 共享

安全下下文: public_content_t
挂载查看帮助使用命令: man nfs

  1. 对于 krb5p 验证加密方式客户端和服务的必需开启 nfs-secure 和 nfs-secure-server 两个服务才能正常通过验证
  2. krb5p 的证书文件权限需要是 440
  3. 版本不同步问题，文件 /etc/sysconfig/nfs 中设置 RPCNFSDARGS="-V 4.2"

    实验过程中未发现此问题，不过还是记录一下以防万一

fstab 文件例子

server0:/protected /mnt/nfssecure nfs defaults,sec=krb5p 0 0

4.对于 Apache 的备注

  1. 对于拒绝某个域的访问可直接使用防火墙禁止他的网段或者使用

    Reuquire not host .baidu.com

  2. 对于 wsgi 程序的配置注意安全上下文

    WSGIScriptAlias / /var/www/webapp/webinfo.wsgi

  3. 对于 ssl 证书的权限要设置为 600

目录权限配置参考表格如下
Require all denied 拒绝所有访问
Require all granted 允许所有访问
Require ip 172.17.10.10 允许172.17.10.10这个地址访问
Require not ip 172.17.10.10 拒绝172.17.10.10这个地址访问
Reuquire host .example.com 允许.example.com这个域访问
Reuquire not host .example.com 拒绝.example.com这个域访问

5.对于 iscsi 的备注

服务端服务:

$ yum install targetcli
$ system enable target
$ targetcli # 进入管理工具
# 建立后备存储库
/> /backstores/block/ create dev=/dev/sun_vg/sun_lv name=server0.iscsi_store
# 创建一个iscsi服务磁盘,磁盘名(iqn.name)题目中有
/> /iscsi/ create iqn.name
# 服务磁盘关联后备存储库
/> /iscsi/iqn.name/tpg1/luns/ create /backstores/block/server0.iscsi_store
# 为客户端建立node允许访问 客户端名字(client.iqn.name)在客户端 (/etc/iscsi/initiatorname.iscsi)获取
/> /iscsi/iqn.name/tpg1/acls/ create client.iqn.name
# 配置监听的ip和端口 如题目中有要求的话
/> /iscsi/iqn.name/tpg1//portals/ create ip_port=3060
# 保存退出
/> exit

客户端服务

$ yum install iscsi-initiator-utils
$ system enable iscsi
$ iscsiadm -m discovery -t sendtargets -p 172.25.0.11
$ iscsiadm -m node -l
$ vim /etc/fstab

fstab 文件例子

UUID="51266761-58f8-48e8-b344-e88792bf79bd" /mnt/data ext4 _netdev 0 0

7.关于 Shell 脚本

一些 关键词语法糖可到 /etc/init.d 目录下查找例子

  1. 脚本1
#!/bin/bash
case $1 in
    redhat)
        echo "fedora"
    ;;
    fedora)
        echo "redhat"
    ;;
    *)
    echo "/root/foo.sh redhat|fedora"
esac
  1. 脚本2
#!/bin/bash
if [ $# -eq 1 ]; then # 首先判断脚本的所有参数数量等于 1 否
    if [ -f "$1" ];then # $1 是否存在且是文件
        while read username
            do
                useradd -s /bin/false $username &> /dev/null
            done < $1 # 以参数 1 文件重定向输入
    else
        echo "Input file not found"
        exit 1
    fi
else
    echo "usage: /root/batchusers userfile"
    exit 2
fi
